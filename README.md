# Healthy AI

Healthy AI refers to an AI-powered application that supports human learning. This involves providing information that is:

- **Accurate:** Correct, up-to-date and references external systems.
- **Foundational:** Directs towards foundational (basic) information.
- **Related:** Can be connected or mapped with one another.
- **Goal-oriented:** Focused on the very next step that solves the problem.

## Mini Examples

The folowing examples start with problems that a user wants to solve. Each problem is followed with directions on how a Healthy AI should provide an answer.

### Accurate

**Problem:** What is the weather in Toronto?

Healthy AI should consult external sources like Weather.com and accurately retrieve the weather conditions in Toronto.

### Foundational

**Problem:** What is the solution to `x + 7 = 10`?

Healthy AI should direct towards foundational Algebra lessons that provide structured steps on solving similar problems. Examples of such lessons include Khan Academy videos or OpenStax textbooks.

### Related

**Problem:** What is an HTML page?

Healthy AI should refer to related concepts like websites and computer languages.

### Goal-oriented

**Problem:** How do you fix a broken camera?

Healthy AI should first inquire about how the camera is broken.
